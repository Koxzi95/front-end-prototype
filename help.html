QAL Library System Help

General Help

If on any of the screens you click a button and a message of 'You do not have the correct system access' appears this means you do not have admin privilages.
When finished using the system return to your home page, by using the return button on every page. Then click Sign Out located in the top right corner.

Login

You should use your provided UID and your personal password. If you have trouble logging in please contact ITMS @:x for help or to create an account. If you have forgotten your password then use the Forgotten Password link on the login screen.
Suspended Borrowers: If you have been suspended from the library for any reason, your login details will be invalid for the time of you suspension and you will see this message 'Login Details Suspended'.

Admin Panel

The buttons on this page will take you to the relevant system section. If for any reason these do not work or are working incorrectly, please contact ITMS @:x

Home Page

The buttons on this page will take you to the relevant library section. If for any reason these do not work or are working incorrectly, please contact ITMS @:x

Book Catalogue

Search: This will allow you to search the entire book catalogue.
Add: This will allow you to add a new book to the catalogue.
Edit: This will allow to you amend a books details in the catalogue.
Delete: This will allow you to delete a book from the system.

Fines

Search: This will allow you to search the entire outstanding fines of all registered library users.
Add: This will allow you to add a new fine to the database.
Edit: This will allow to you amend fine details in the database.
Delete: This will allow you to delete a fine from the system.

Loans

Search: This will allow you to search for existing loans of all registered library users.
Edit: This will allow to you amend loan details in the database.
Add: This will allow you to add a new loan to the database.
Delete: This will allow you to delete a loan from the system.

New Loan

Search Book: This button will allow you to search for a book.
Search User: This allows you to search for a specific user.
Select Return Date: This wil allow you to pick a date to return the book, no longer than 3 weeks
Return: This will take you back to your main page.

Payments

Search: This will allow you to search the entire payment history of all registered library users.
Add: This will allow you to add a new payment to the database.
Edit: This will allow to you amend payment details in the database.

Reservations

Search: This will allow you to search all current reservations of all registered library users.
Add: This will allow you to add a new reservation to the database.
Edit: This will allow to you amend reservation details in the database.
Delete: This will allow you to delete a reservation from the system.

User Management

Search: This will allow you to search all registered library users.
Add: This will allow you to add a new user to the database.
Edit: This will allow to you amend user details in the database.
Delete: This will allow you to delete a user from the system.
