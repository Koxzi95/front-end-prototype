General Help

If on any of the screens you click a button and a message of 'You do not have the correct system access' appears this means you do not have admin privilages.
When finished using the system return to your home page, by using the return button on every page. Then click Sign Out located in the top right corner.


Login

You should use your provided UID and your personal password. If you have trouble logging in please contact ITMS @:x for help or to create an account. If you have forgotten your password then use the Forgotten Password link on the login screen.
Suspended Borrowers: If you have been suspended from the library for any reason, your login details will be invalid for the time of your suspension and you will see this message 'Login Details Suspended'.

Book Catalogue

Search: This will allow you to search the entire book catalogue.
Add: This will allow you to add a new book to the catalogue.
Edit: This will allow to you amend a books details in the catalogue.
Delete: This will allow you to delete a book from the system.


Fines

Search: This will allow you to search the entire outstanding fines of all registered library users.
Add: This will allow you to add a new fine to the database.
Edit: This will allow to you amend fine details in the database.
Delete: This will allow you to delete a fine from the system.


Loans

Search: This will allow you to search for existing loans of all registered library users.
Edit: This will allow to you amend loan details in the database.
